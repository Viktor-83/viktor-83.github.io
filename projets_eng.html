Tutored & Personal Projects

NaviCampus Project (3D Navigation)

Interactive 3D mapping and navigation web application modeling the Aix-Marseille University Institute of Technology (IUT) for Networks and Telecommunications.

  • ▹
    3D Rendering: Multi-floor building modeling using the Three.js JavaScript library.
  • ▹
    Algorithm: Implementation of A* pathfinding to calculate the shortest route between rooms, including dynamic handling of stairs and elevators.
  • ▹
    Accessibility: Integration of a reduced mobility (PRM) system that restricts calculated routes strictly to elevators.

Escape from Tarkov Server Deployment (SPT / Headless)

Setup of an optimized server infrastructure to host a private multiplayer session of the game Escape From Tarkov via the SPT (Single Player Tarkov) emulator.

  • ▹
    Headless Environment: Configuration of a dedicated server without a graphical interface (command line) to free up processing power (RAM/CPU) for game hosting.
  • ▹
    Emulation & Modding: Installation of the SPT framework (which emulates the complex backend architecture of the original game) and integration of network mods to allow private cooperation.
  • ▹
    Network Administration: Connectivity management via specific port forwarding, firewall configuration, and access security for remote players.
  • ▹
    Server Maintenance: Full remote administration via SSH and centralized management of player database backups.

Mobile Chat Application

Design and development of a client-server instant messaging solution, focusing on secure exchanges.

  • ▹
    Server Architecture: Implementation of a robust TCP server in Python for managing multiple connections.
  • ▹
    Cryptography: Securing data streams by integrating the cryptography/Fernet module for message encryption.
  • ▹
    Client Development: Creation of the mobile interface and communication logic with the Kivy framework.
  • ▹
    Mobile Deployment: Compilation of the project into an APK application using the Buildozer tool for installation on an Android environment.
  • ▹
    Management & DevOps: Implementation of authentication via SQLite database, detailed documentation, and use of GitHub/GitLab for versioning.

Enterprise Aggregation Solution

Creation of an automated platform to collect, process, and present data from external feeds in a centralized manner.

  • ▹
    Data Collection: Development of a Python script dedicated to querying and aggregating multiple RSS feeds.
  • ▹
    Processing & Storage: Structuring and long-term saving of obtained data in interoperable standard formats (JSON, YAML).
  • ▹
    Web Interface: Formatting and dynamic presentation of collected information on a modern HTML5 page.
  • ▹
    System Administration: Deployment and full management of a web server machine and the aggregator in a secure Linux environment (SSH).

Network & IoT Architecture

Construction of a network infrastructure combining advanced server monitoring and data collection from physical sensors (IoT).

  • ▹
    Remote Administration: Establishing a secure SSH connection to a remote server from a Linux VM, including specific configuration of the hosts/config file.
  • ▹
    IoT Telemetry: Configuration of a Raspberry Pi microcomputer as a message broker (Mosquitto) for weather data retrieval.
  • ▹
    Monitoring: Setup of comprehensive dashboards with Grafana to ensure continuous monitoring of VMs and the host server's status.
  • ▹
    Graphic Visualization: Integration and dynamic display of real-time data via HTML5/CSS graphs.
  • ▹
    Versioning: Repository, documentation, and continuous management of the project's source code using the GitLab platform.
