Projet NaviCampus (Navigation 3D)

Application web interactive de cartographie et de navigation 3D, modélisant l'IUT Réseaux et Télécommunications d'Aix-Marseille.

  • ▹
    Rendu 3D : Modélisation du bâtiment sur plusieurs étages avec la bibliothèque JavaScript Three.js.
  • ▹
    Algorithmique : Implémentation du pathfinding A* pour calculer le trajet le plus court entre deux salles, avec gestion dynamique des escaliers et ascenseurs.
  • ▹
    Accessibilité : Intégration d'un système de mobilité réduite (PMR) qui recalcule les itinéraires exclusivement via les ascenseurs.

Serveur Escape from Tarkov (SPT / Headless)

Mise en place d'une infrastructure serveur optimisée pour héberger une session multijoueur privée du jeu Escape From Tarkov via l'émulateur SPT (Single Player Tarkov).

  • ▹
    Environnement Headless : Configuration d'un serveur dédié sans interface graphique (en ligne de commande) afin de libérer la puissance de calcul (RAM/CPU) pour l'hébergement du jeu.
  • ▹
    Emulation & Modding : Installation du framework SPT (qui émule l'architecture backend complexe du jeu d'origine) et intégration de mods réseau pour permettre la coopération privée.
  • ▹
    Administration Réseau : Gestion de la connectivité via l'ouverture des ports spécifiques (Port Forwarding), configuration du pare-feu, et sécurisation des accès pour les joueurs distants.
  • ▹
    Maintenance Serveur : Administration complète à distance via SSH et gestion centralisée des sauvegardes de la base de données des joueurs.

Application de chat mobile

Conception et développement d'une solution de messagerie instantanée client-serveur, axée sur la sécurisation des échanges.

  • ▹
    Architecture Serveur : Implémentation d'un serveur TCP robuste en Python pour la gestion de connexions multiples.
  • ▹
    Cryptographie : Sécurisation des flux de données via l'intégration du module cryptography/Fernet pour le chiffrement des messages.
  • ▹
    Développement Client : Création de l'interface mobile et de la logique de communication avec le framework Kivy.
  • ▹
    Déploiement Mobile : Compilation du projet en application APK avec l'outil Buildozer pour une installation sur environnement Android.
  • ▹
    Gestion & DevOps : Mise en place d'une authentification via base de données SQLite, documentation détaillée et utilisation de GitHub/GitLab pour le versioning.

Solution d'agrégation d'entreprise

Création d'une plateforme automatisée permettant de collecter, de traiter et de restituer des données issues de flux externes de manière centralisée.

  • ▹
    Collecte de Données : Développement d'un script en Python dédié à l'interrogation et l'agrégation de multiples flux RSS.
  • ▹
    Traitement & Stockage : Structuration et sauvegarde pérenne des données obtenues sous formats standards interopérables (JSON, YAML).
  • ▹
    Interface Web : Mise en forme et présentation dynamique des informations collectées sur une page HTML5 moderne.
  • ▹
    Administration Système : Déploiement et gestion complète d'une machine serveur web et de l'agrégateur sous un environnement Linux sécurisé (SSH).

Architecture Réseau & IoT

Construction d'une infrastructure réseau combinant la supervision avancée de serveurs et la remontée de données issues de capteurs physiques (IoT).

  • ▹
    Administration Distante : Établissement d'une connexion SSH sécurisée vers un serveur distant depuis une VM Linux, incluant la configuration spécifique du fichier hosts/config.
  • ▹
    Télémétrie IoT : Configuration d'un nano-ordinateur Raspberry Pi en tant que broker de messagerie (Mosquitto) pour la récupération de données météorologiques.
  • ▹
    Monitoring : Mise en place de tableaux de bord complets avec Grafana pour assurer la surveillance continue des VMs et de l'état du serveur hôte.
  • ▹
    Visualisation Graphique : Intégration et affichage dynamique des données en temps réel via des graphiques en HTML5/CSS.
  • ▹
    Versioning : Dépôt, documentation et gestion continue du code source du projet en utilisant la plateforme GitLab.
